# FoodieGo-Food-Delivery-App
A modern food delivery mobile app UI/UX design created in figma.

## Overview

A modern food delivery mobile app UI/UX design created in Figma. The app is designed to provide a smooth and user-friendly food ordering experience, from browsing restaurants to placing an order.

## Features

- Browse restaurants and menu items
- View food details
- Add and manage items in cart
- Secure checkout process
- Order confirmation screen
- Clean and modern user interface

## Tools Used
- Figma
  
## Design Process

- Wireframing
- UI Design
- Prototyping

## Screens Included

- Splash Screen
- Home Screen
- Restaurant Details Screen
- Cart Screen
- Checkout Screen
- Order Success Screen

## Concepts Learned

- User-Centered Design
- Visual Hierarchy
- Layout and Spacing
- UI Design Principles
- Mobile App Design
- Interactive Prototyping
